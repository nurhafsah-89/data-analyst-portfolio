# 📊 SLA Performance Dashboard (Power BI)

## 🔒 Code & Data Availability

Due to the use of API keys and sensitive operational data, the full implementation and datasets are not included in this public repository.
A sanitized version of the code and sample outputs can be prepared upon request.

💻 Demo or walkthrough can be provided if needed.


## 📌 Overview
Developed a comprehensive Weekly Trend Power BI dashboard using FreshDesk data, featuring Executive Overview, Commodity-level SLA Performance, Overall SLA Performance, and Ticket Trend Analysis.


---

## ⚙️ Key Features
- Executive-level summary of SLA performance
- Commodity-level SLA tracking
- Overall SLA performance monitoring
- Ticket trend analysis (weekly trends)
- Interactive filtering and drill-down capability

---

## 🛠 Tools & Technologies
- Power BI
- DAX
- Data Modeling
