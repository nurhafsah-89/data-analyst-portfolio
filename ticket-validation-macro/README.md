# ⚙️ Excel Automation (VBA & Macros)

## 📌 Project 1: Ticket Validation Macro

### 📊 Overview
Developed an Excel-based Ticket Validation Macro to automate the weekly ticket validation process. The solution improves validation speed, reduces manual dependency, and enhances data consistency for SLA performance reporting.

### ⚙️ Key Features
- Automated ticket validation using macro logic
- Rule-based classification (priority, ticket type)
- Status updates and validation checks
- Highlighting for filtered or updated records

### 🛠 Tools & Technologies
- Excel VBA
- Macros

### 📊 Business Impact
- Reduced manual effort in weekly validation
- Improved data accuracy and consistency
- Increased efficiency in SLA reporting
