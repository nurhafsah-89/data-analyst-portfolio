# 🚀 Project Vista – Automation Solution

## 📌 Overview
Developed an automation solution to extract and transform unstructured shipment data (PDF/Image) into structured Excel format, improving efficiency and reducing manual processing effort.

---

## ⚙️ Key Features
- Automated PDF/Image to Excel extraction
- Data cleaning and transformation using Python (Pandas)
- Regex-based text parsing for structured data mapping
- Batch processing for handling multiple files efficiently

---

## 🛠 Tools & Technologies
- Python (Pandas, Regex)
- Playwright (Web Automation)
- Excel

---

## 📊 Business Impact
- Reduced manual data entry effort
- Improved data accuracy and consistency
- Enabled faster processing of shipment data
